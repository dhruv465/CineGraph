# Cinematic Video Portfolio Template

A visually stunning, modern portfolio template designed for video producers, cinematographers, and creative studios. Built with React and Vite.

## Features

- **Dynamic Layout:** A responsive design that looks great on all devices.
- **Physics-Based UI:** Interactive elements (like the category pills in the footer) powered by `matter-js`.
- **Smooth Animations:** Integrated CSS animations and scroll-based reveal effects.
- **Easy Customization:** Replace placeholder text ("John Doe") and images to make it your own.

## Getting Started

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Start the development server:**
   ```bash
   npm run dev
   ```

3. **Build for production:**
   ```bash
   npm run build
   ```

## Customization

- **Personal Information:** Search the codebase for generic placeholders like "John Doe", "dummy@example.com", and update them with your details.
- **Images:** Replace placeholder images in the `public` directory.
- **Content:** Update the text in components like `Hero.tsx`, `About.tsx`, and `ClientsServices.tsx`.
- **Videos:** Update the YouTube video IDs in the `categories` array within `Footer.tsx`.
